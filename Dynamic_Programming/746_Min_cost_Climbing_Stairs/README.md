＃難度： easy

＃題目說明： 
- 爬梯子，從第零階開始爬，爬到頂
- 每一層都可以選擇爬 1 或者 2 階，但是需要付出 cost array 顯示的價錢

＃例子       
- cost  =     [10, 15, 20]
- 階數  =  0    1   2   3   4


- 表示爬上第一階需要花費 10，以此類推
- 要爬到第 4 階，最少需要花費 15
1. 0 階 -> 2 階，花費 0
2. 2 階 -> 4 階，花費 15

＃想法
- 用一個 dp array index n 表示爬到 n + 1 階最少需要多少 cost
- 每次更新  dp[n] 時，表示可能爬到 dp[n] 的只有 dp[n-1] 跟  dp[n-2] 的加上各自的  cost （因為每次都可以選擇爬 1 或 2 階）
