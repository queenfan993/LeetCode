＃難度： Medium

＃題目說明： 
- 給一個 int array nums 和一個目標數字 0
- 回傳在 nums 中所有三個數字合可以等於 0 的組合 
- 數字可能重複

＃ Constraints
- 3 <= nums.length <= 3000
- -10^5 <= nums[i] <= 10^5

＃例子
- Input: nums = [-1,0,1,2,-1,-4]
- Output: [[-1,-1,2],[-1,0,1]]

＃想法
- 先 sort nums
- 用一個迴圈從 i = 0 開始，在迴圈中每次都有兩個指標
- 從 i + 1 和 length - 1 開始往中間
- 判斷三數是否大於小於 0，小於就移動左指標向右，反之亦然
 
＃ 時間複雜度
- 近似兩層迴圈 O(n^2)
